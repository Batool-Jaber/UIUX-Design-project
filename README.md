## 📌 Project 1: Smart Water Bottle App

### Problem
Users struggle with:
- Keeping water clean
- Maintaining temperature
- Avoiding plastic waste
- Regular cleaning inconvenience

### Solution
I designed an app for a smart self-cleaning bottle that:
- Tracks cleaning cycles
- Controls temperature
- Provides a seamless user experience

### My Role
- Full UI/UX Design (independent project)
- Prototyping
- Self-learning under time pressure

### Key Features
- Clean and minimal interface
- Easy control of bottle features
- Focus on user convenience

### Tools Used
- Figma

## 📌 Project 2: Modepack App (Modular Laptop Bag)

### Problem
Traditional laptop bags are often boring, heavy, and lack flexibility.  
Users need a stylish yet practical bag that adapts to their daily outfits without buying multiple bags.

### Solution
I designed a mobile app that allows users to:
- Customize their bag with interchangeable parts
- Match colors with their outfits
- Enjoy a smooth and simple user experience

### My Role
- UI/UX Design
- Prototyping
- Improving usability and layout
- Team collaboration under tight deadlines

### Key Features
- Simple and intuitive interface
- Fast customization process
- Modern and stylish UI

### Tools Used
- Figma
